Overview

This is just a simple prototype for TENA Home Delivery.
It consist of User Interface only - no database integration.

The goal is to give a rough idea of what type of alternatives could be developed to replace the current SAP Portal solution.

This one was mostly done with modern web technologies, HTML5 related.
By using these type of tools one can guarantee:

  • 1) Truly RWD (responsive web design). This means the web-site will behave seamlessly on any browser, screen or device.
  • 2) Easy/fast to develop. No compilation, libraries or complex dependencies.


    • I have created 2 scenarios. "Admin" and "End User".
